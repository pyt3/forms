# Prompt สำหรับสร้าง Poster: คู่มือติดตั้งและใช้งานแอพย้ายเตียง

## บทบาทของ AI

คุณคือ AI นักออกแบบ Poster สำหรับสื่อสารวิธีติดตั้งและใช้งานแอพภายในโรงพยาบาล ให้ออกแบบ Poster ภาษาไทยที่อ่านง่าย เหมาะสำหรับเจ้าหน้าที่หน้างาน ใช้บนกระดาษ A4 แนวตั้ง หรือภาพแนวตั้งสำหรับส่งในไลน์กลุ่ม

## ชื่อระบบ

**แอพย้ายเตียง**

## คำอธิบายสั้น

แอพย้ายเตียงเป็นเว็บแอพแบบ PWA สำหรับบันทึกการย้ายเตียงหรืออุปกรณ์ในโรงพยาบาล โดยสแกน QR Code ที่ติดกับเตียง เลือกหรือกรอกข้อมูลห้องปลายทาง ใส่ชื่อผู้บันทึก แล้วส่งข้อมูลไปยังระบบ Google Apps Script แอพรองรับการติดตั้งบนมือถือ ใช้งานกล้องเพื่อสแกน QR Code เปิดไฟฉายได้ในอุปกรณ์ที่รองรับ และสามารถบันทึกข้อมูลไว้ชั่วคราวเมื่อออฟไลน์ จากนั้นซิงค์อัตโนมัติเมื่อกลับมาออนไลน์

## จุดเด่นที่ต้องสื่อใน Poster

- ติดตั้งเป็นแอพบนมือถือได้ผ่านปุ่ม **ติดตั้ง**
- เปิดแอพแล้วสแกน QR Code ของเตียงได้ทันที
- หากสแกนไม่ได้ สามารถกด **กรอกรหัสเอง**
- แสดงข้อมูลอุปกรณ์จากไฟล์ `equipment_list.xlsx`
- กรอกเลขห้องและชื่อผู้บันทึกก่อนบันทึก
- ชื่อผู้บันทึกจะถูกจำไว้ในเครื่องเพื่อใช้งานครั้งถัดไป
- ใช้งานได้ทั้งออนไลน์และออฟไลน์
- เมื่อออฟไลน์ ระบบจะเก็บคิวไว้ในเครื่อง และส่งข้อมูลอัตโนมัติเมื่อออนไลน์
- มีสถานะ **ออนไลน์ / ออฟไลน์** ให้เห็นชัดเจน
- มีแจ้งเตือนเมื่อแอพมีเวอร์ชันใหม่

## คู่มือการติดตั้งแอพสำหรับผู้ใช้งาน

### Android / Chrome

1. เปิดลิงก์แอพย้ายเตียงผ่าน Google Chrome
2. รอให้หน้าแอพโหลดเสร็จ
3. กดปุ่ม **ติดตั้ง** ที่มุมบนของแอพ ถ้าปุ่มปรากฏ
4. ยืนยันการติดตั้งตามหน้าจอของ Chrome
5. หลังติดตั้งเสร็จ จะมีไอคอนแอพ **ย้ายเตียง** บนหน้าจอมือถือ
6. เปิดใช้งานจากไอคอนได้เหมือนแอพทั่วไป

### iPhone / Safari

1. เปิดลิงก์แอพย้ายเตียงผ่าน Safari
2. กดปุ่มแชร์ของ Safari
3. เลือก **เพิ่มไปยังหน้าจอโฮม**
4. ตั้งชื่อเป็น **ย้ายเตียง**
5. กด **เพิ่ม**
6. เปิดแอพจากไอคอนบนหน้าจอโฮม

### สิทธิ์ที่ต้องอนุญาต

- อนุญาตให้ใช้ **กล้อง** เพื่อสแกน QR Code
- ใช้ผ่าน HTTPS หรือ localhost เท่านั้น เพื่อให้เบราว์เซอร์อนุญาตการใช้งานกล้อง
- หากต้องใช้ไฟฉาย อุปกรณ์และเบราว์เซอร์ต้องรองรับการควบคุมแฟลช

## คู่มือการใช้งานแอพ

### ขั้นตอนหลัก

1. เปิดแอพ **ย้ายเตียง**
2. ตรวจสอบสถานะด้านบนว่าเป็น **ออนไลน์** หรือ **ออฟไลน์**
3. หันกล้องไปที่ QR Code ของเตียงหรืออุปกรณ์
4. เมื่อสแกนสำเร็จ แอพจะแสดงหน้ากรอกข้อมูล
5. ตรวจสอบข้อมูลอุปกรณ์ที่แสดง เช่น ID CODE และรูปภาพถ้ามี
6. กรอก **เลขห้อง** ปลายทาง
7. กรอก **ชื่อผู้บันทึก**
8. กดปุ่ม **บันทึก**
9. หากบันทึกสำเร็จ ระบบจะแสดงข้อความ **บันทึกสำเร็จ!**
10. แอพจะกลับไปหน้าเปิดกล้องเพื่อสแกนรายการถัดไป

### กรณีสแกน QR Code ไม่ได้

1. กดปุ่ม **กรอกรหัสเอง**
2. ระบุเลขเตียงหรือรหัสที่ต้องการค้นหา
3. กด **ตกลง**
4. หากพบข้อมูลในรายการอุปกรณ์ แอพจะเข้าสู่หน้ากรอกข้อมูล
5. กรอกเลขห้องและชื่อผู้บันทึก แล้วกด **บันทึก**

### กรณีใช้งานในพื้นที่แสงน้อย

1. เมื่อเปิดกล้องแล้ว ถ้าอุปกรณ์รองรับ จะมีปุ่มไฟฉายแสดงขึ้น
2. กดปุ่มไฟฉายเพื่อเปิดหรือปิดแฟลช
3. เมื่อเข้าสู่หน้ากรอกข้อมูล ระบบจะซ่อนไฟฉายและปิดแฟลชโดยอัตโนมัติ

### การใช้งานแบบออฟไลน์

1. หากไม่มีอินเทอร์เน็ต สถานะด้านบนจะแสดงเป็น **ออฟไลน์**
2. ผู้ใช้ยังสามารถสแกนและบันทึกข้อมูลได้
3. ระบบจะแสดงข้อความ **บันทึกในโหมดออฟไลน์**
4. ข้อมูลจะถูกเก็บไว้ในเครื่องชั่วคราว
5. เมื่อกลับมาออนไลน์ ระบบจะส่งข้อมูลค้างส่งให้อัตโนมัติ

### การอัปเดตแอพ

1. หากมีเวอร์ชันใหม่ แอพจะแสดงแถบแจ้งเตือน **มีเวอร์ชันใหม่พร้อมใช้**
2. กดปุ่ม **อัปเดตเลย**
3. แอพจะโหลดเวอร์ชันล่าสุดและพร้อมใช้งานต่อ

## ข้อมูลทางเทคนิคสำหรับใส่ใน Poster แบบย่อ

- ประเภทแอพ: Web App / PWA
- ไฟล์หลัก: `index.html`
- ไฟล์ข้อมูลอุปกรณ์: `equipment_list.xlsx`
- ระบบบันทึกข้อมูล: Google Apps Script
- รองรับ Offline Queue ผ่าน Local Storage
- รองรับ Service Worker สำหรับ cache และการอัปเดต
- ไลบรารีที่ใช้: Tailwind CSS, jQuery, Lucide Icons, QR Scanner, SheetJS XLSX

## โทนภาพที่ต้องการ

- โทนสีหลัก: น้ำเงินเข้ม, ฟ้า, ขาว, เขียวสำหรับความสำเร็จ, เหลืองสำหรับออฟไลน์
- อารมณ์ภาพ: สะอาด ทันสมัย ใช้งานจริงในโรงพยาบาล อ่านง่ายสำหรับเจ้าหน้าที่
- ใช้ไอคอนประกอบ เช่น มือถือ, QR Code, กล้อง, เตียง, Wi-Fi, cloud sync, check mark
- จัดวางเป็นขั้นตอนชัดเจน ไม่ใส่ข้อความแน่นเกินไป
- ให้ความสำคัญกับขั้นตอน **ติดตั้ง > สแกน > กรอกห้อง > บันทึก > ซิงค์ออฟไลน์**

## โครงสร้าง Poster ที่แนะนำ

### ส่วนที่ 1: หัว Poster

- ชื่อ: **แอพย้ายเตียง**
- คำโปรย: **บันทึกการย้ายเตียงได้รวดเร็ว ด้วย QR Code**
- ใส่ภาพมือถือที่แสดงหน้าสแกน QR Code

### ส่วนที่ 2: วิธีติดตั้ง

ใช้กล่องขั้นตอน 3 ขั้น:

1. เปิดลิงก์แอพใน Chrome หรือ Safari
2. กด **ติดตั้ง** หรือ **เพิ่มไปยังหน้าจอโฮม**
3. เปิดใช้งานจากไอคอน **ย้ายเตียง**

### ส่วนที่ 3: วิธีใช้งาน

ใช้กล่องขั้นตอน 5 ขั้น:

1. เปิดแอพ
2. สแกน QR Code ที่เตียง
3. ตรวจสอบ ID CODE
4. กรอกเลขห้องและชื่อผู้บันทึก
5. กด **บันทึก**

### ส่วนที่ 4: ฟีเจอร์สำคัญ

- กรอกรหัสเองได้เมื่อสแกนไม่ได้
- เปิดไฟฉายได้ในเครื่องที่รองรับ
- บันทึกออฟไลน์ได้
- ซิงค์ข้อมูลอัตโนมัติเมื่อออนไลน์
- แจ้งเตือนเมื่อมีเวอร์ชันใหม่

### ส่วนที่ 5: หมายเหตุ

ใส่ข้อความเล็กด้านล่าง:

**โปรดอนุญาตการใช้งานกล้อง และตรวจสอบว่าข้อมูลเลขห้องถูกต้องก่อนกดบันทึก**

## Prompt พร้อมใช้สำหรับ AI สร้าง Poster

สร้าง Poster ภาษาไทยแนวตั้ง ขนาด A4 สำหรับคู่มือติดตั้งและใช้งาน **แอพย้ายเตียง** ในโรงพยาบาล โทนสีสะอาด ทันสมัย ใช้สีน้ำเงินเข้ม ฟ้า ขาว และเขียว เน้นอ่านง่ายสำหรับเจ้าหน้าที่หน้างาน

หัวข้อใหญ่: **คู่มือติดตั้งและใช้งานแอพย้ายเตียง**

คำโปรย: **สแกน QR เตียง กรอกห้อง บันทึกได้ทันที ใช้งานได้ทั้งออนไลน์และออฟไลน์**

จัด Poster เป็น 5 ส่วน:

1. ส่วนหัว แสดงชื่อ **แอพย้ายเตียง** พร้อมภาพมือถือหน้าสแกน QR Code และไอคอนเตียง
2. วิธีติดตั้ง: เปิดลิงก์แอพใน Chrome หรือ Safari, กด **ติดตั้ง** หรือ **เพิ่มไปยังหน้าจอโฮม**, เปิดใช้งานจากไอคอน **ย้ายเตียง**
3. วิธีใช้งาน: เปิดแอพ, สแกน QR Code ที่เตียง, ตรวจสอบ ID CODE, กรอกเลขห้องและชื่อผู้บันทึก, กด **บันทึก**
4. ฟีเจอร์สำคัญ: กรอกรหัสเองได้, เปิดไฟฉายได้, บันทึกออฟไลน์ได้, ซิงค์อัตโนมัติเมื่อออนไลน์, แจ้งเตือนเวอร์ชันใหม่
5. หมายเหตุด้านล่าง: **โปรดอนุญาตการใช้งานกล้อง และตรวจสอบเลขห้องก่อนกดบันทึก**

ใช้ไอคอนประกอบ เช่น QR Code, กล้อง, เตียงโรงพยาบาล, มือถือ, Wi-Fi, cloud sync, check mark จัดวางเป็นขั้นตอนชัดเจน มีพื้นที่ว่างพอดี ไม่ใส่ข้อความแน่นเกินไป ให้ภาพรวมดูน่าเชื่อถือและเหมาะกับระบบภายในโรงพยาบาล
